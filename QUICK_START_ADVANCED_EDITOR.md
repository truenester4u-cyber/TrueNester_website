## ‚ö° Quick Actions

### Text Formatting
1. Select text
2. Click **Bold**, **Italic**, or **Underline**
3. Or use shortcuts: `Ctrl+B`, `Ctrl+I`, `Ctrl+U`

### Change Font
1. Click **Font Family** dropdown (top left)
2. Choose from 8 fonts (Inter, Arial, Times New Roman, etc.)

### Change Font Size
1. Click **Font Size** dropdown
2. Select 12px to 72px

### Add Colors
1. Click **Palette** icon (color picker)
2. Choose from 16 text colors
3. Click **Highlighter** icon for background colors

### Insert Link
1. Click **Insert > Link** or press `Ctrl+K`
2. Choose link type: External, Email, or Phone
3. Enter URL and optional link text
4. Check "Open in new tab" if needed
5. Click **Insert Link**

### Insert Table
**Option A - Custom Table**:
1. Click **Insert > Table** or press `Ctrl+Shift+T`
2. Enter rows and columns
3. Click **Insert Table**

**Option B - Use Template**:
1. Click **Insert > Table**
2. Go to **Templates** tab
3. Choose:
   - **Amenities List** (Swimming Pool, Gym, etc.)
   - **Price Comparison** (Unit types and prices)
   - **Floor Plans** (Bedrooms, bathrooms, sizes)

### Edit Tables
1. Click inside a table cell
2. Switch to **TABLE** tab
3. Use buttons to:
   - Add rows above/below
   - Add columns left/right
   - Delete rows/columns
   - Merge cells
   - Delete entire table

### Upload Images
1. Click **Insert > Image**
2. Go to **Upload** tab
3. Drag & drop image or click to browse
4. Click **Upload & Insert**
5. Image uploads to Supabase Storage automatically

### Embed Videos
1. Copy YouTube or Vimeo URL
2. Click **Insert > Video**
3. Paste URL
4. Click **Embed Video**
5. Video appears as responsive player

### Add Emojis
1. Click **Insert > Emoji**
2. Choose from 18 property emojis (üè†, üè¢, üõèÔ∏è, etc.)

### Insert Special Characters
1. Click **Insert > Symbols**
2. Choose from 20+ symbols
3. Includes currency: **AED**, $, ‚Ç¨, ¬£, ¬•, ‚Çπ

## üîç Advanced Features

### Live Preview
1. Click **Show Preview** button (top right)
2. See split-screen: Editor | Preview
3. Preview updates in real-time
4. Perfect for checking formatting

### Find & Replace
1. Press `Ctrl+F` or click **Tools > Find & Replace**
2. Enter search term
3. Enter replacement text
4. Click **Replace All**

### Version History
1. Click **History** button (top bar)
2. See all saved versions with timestamps
3. Shows who made each edit (user avatar + name)
4. Click any version to restore it

### Auto-save
- Editor saves every **20 seconds** automatically
- Watch the status indicator:
  - üî¥ Red dot = Unsaved changes
  - "Saving..." = Currently saving
  - "Saved" = All changes saved
- See timestamp: "Last saved: X minutes ago"

### Export Content
1. Click **Export** dropdown (top right)
2. Choose format:
   - **Copy HTML**: Copy to clipboard
   - **Download HTML**: Save as .html file
   - **Download Text**: Save as .txt file

## üêõ Troubleshooting

### Editor Not Showing
1. Clear browser cache (Ctrl+Shift+Delete)
2. Hard refresh (Ctrl+F5)
3. Check console for errors (F12)

### Image Upload Failing
1. Check file size (max 10MB)
2. Verify Supabase Storage permissions
3. Ensure `property-images` bucket exists

### Auto-save Not Working
1. Check network connection
2. Verify you're logged in
3. Ensure propertyId exists (for edit mode)

### Version History Empty
- Versions appear after first auto-save (20 seconds)
- Only saved when editing existing properties
- New properties create versions after first save
